** Nat

Description
- Represents a natural number: 0, 1, …

Constructors
- nat : Integer → Result(Nat) :≡
  - wrapping is necessary because int must be ≥ 0.

Eliminator
- elim : (Integer → C) → Nat → C

Interface
- int : Nat → Integer :≡ elim(identity)
- + : Nat Nat → Nat :≡
  - n1 n2 ↦ nat(n1#int() + n2#int())
