* Index

This file is an index pointing to all documentations. You are reading it using
Emacs. The section below shows basic Emacs commands: how to close it, switch buffers
and so on. To reach a documentation node, just click on a link. A link starts with =→=.

Assuming Guix is installed and you have access to QEMU/KVM, then start the
environment with:

#+begin_src bash :eval no
./start-dev
#+end_src

** Emacs

> How to close Emacs?

Type: control and c, then control and x.
It is usually written: C-c C-x.


> How to close a window?

C-x 0


> How to keep only the current window?

C-x 1


> How to move pointer to another window?

C-x o


> How to list all opened buffers?

C-x C-b
