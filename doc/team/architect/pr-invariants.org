Before sending a PR, check that:
- [ ] The code compiles cleanly, see: [[ref:b80c22c9-2eed-4eea-a469-136e7666ea96][Makefile]].

- [ ] The code is formatted, see: [[ref:b80c22c9-2eed-4eea-a469-136e7666ea96][Makefile]].

- [ ] All tests pass locally, see: [[ref:b80c22c9-2eed-4eea-a469-136e7666ea96][Makefile]].

- [ ] Elixir [[ref:9f24d713-ff38-4659-aff9-d3eb36482109][conventions]] are satisfied.

- [ ] If needed, the [[ref:94a78f42-9098-4796-90c1-725e20a07fbd][architecture]] has been updated.

- [ ] Your iteration reflects the time spent.

- [ ] Your iteration contains remarks, your notes, ideas, or anything that can make
  the next iteration more efficient.
